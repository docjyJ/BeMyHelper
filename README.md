# Lancé BeMyHelper

Cette aplication nécésite un serveur suportant les servlet.

## lancé l'apli
Compilé et récupérer le .war :


```sh
mvn clean install
```


Plus qu'à le déployé sur glassfish ou tomcat par exemple.
